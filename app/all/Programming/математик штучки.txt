 например число 5 его Факториал это 120. Так как 1 * 2 * 3 * 4 * 5 = 120

так можно найти факториал с помощью рекурсии
function fac (n) {
  if ( n === 1) {
      return n;
  } else {
    return n * fac(n - 1);
  }
}

console.log(fac(5));

и с помощью цикла
let total = 1;  создаем общую переменную
function fac (n) {
    for (let i = 0; i < n; i++) { в цикле говорим, что пока i меньше нашего числа, будем работать и i++
        total *= (n - i);
         }
    return total;
}

console.log(fac(5));

че происходит. Значит код чекает такой i меньше чем n значит идем дальеш в total присваиваем n - i.
1: итерации это будет 5 - 0(i) = 5 * total = 5  
2: будет 5 - 1(i) = 4 * total(5) = 20;   в кажоый итерации наша i прибавляется  
3: будет так 5 - 2(i) = 3 * total(20) = 60
4: в нашей i уже (3) 5 - 3 = 2 * total = 120;
5: далее в i(4) 5 - 4 = 1 * total(120) = 120. 
все i = 5 конец
